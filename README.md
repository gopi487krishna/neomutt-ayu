## Installation
---
Include `colors/ayu-dark-256.neomuttrc` as part of your `~/.mutt/muttrc` file.

    $ mkdir -p ~/.mutt
    $ cd ~/.mutt
    $ git clone https://github.com/gopi487krishna/neomutt-ayu.git
    $ echo 'source ~/.mutt/neomutt-ayu/colors/ayu-dark-256.neomuttrc' >>~/.mutt/muttrc
